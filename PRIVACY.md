# Privacy Overview

This repository contains Linux/Ubuntu client release documentation.

A formal independent third-party security audit has not yet been completed.

The Linux client may process VPN profile/subscription configuration locally, generated sing-box/Xray configs, routing settings and runtime logs.

The client should not publish subscription links, profile credentials, UUIDs, tokens or logs without user action and redaction.

No traffic resale, traffic injection or ad injection is intended by project policy. This repository does not contain enough source code to independently prove that behavior for every release artifact.

## Data Retention

This repository does not define server-side retention policy. See service-level privacy policy.

Local retention depends on the installed Linux client and package behavior.
